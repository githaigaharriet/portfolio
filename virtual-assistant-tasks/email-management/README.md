## 📧 Email Management System – Sample Project

### 🧾 Overview  
This project demonstrates how I manage and organize emails for a busy entrepreneur. It includes categorization, prioritization, tracking, and deadline management to ensure efficient communication and task handling.

### 🏢 Industry  
Virtual Assistant / Administrative Support 

### 🛠️ Tools Used  
- Notion  

### 📌 Project Type  
Sample Portfolio Project  

### ✅ Key Features  
- Email categorization (Client, Urgent, Newsletter, Internal)  
- Priority tracking (High, Medium, Low)  
- Status tracking (Pending, In Progress, Completed)  
- Action-based workflow management  
- Deadline tracking with reminders
